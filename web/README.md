## 📰 博客模块 (New)

新增的博客模块集成了内容展示与交互功能，旨在提供沉浸式的阅读体验。

### 核心功能
*   **多维筛选**: 
    *   **搜索**: 实时全文搜索（支持标题和摘要）。
    *   **分类**: 支持二级分类（如技术/前端、设计/UI）。
    *   **标签**: 多选标签过滤，精准定位内容。
*   **沉浸式阅读**:
    *   **每日推荐**: 顶部大屏展示推荐文章，包含作者、阅读时间和摘要。
    *   **瀑布流/网格布局**: 美观的卡片式文章列表，带有封面的悬停微交互。
*   **交互体验**:
    *   **无缝滚动**: 全屏滚动体验，分页组件底部居中。
    *   **入场动画**: 列表项采用交错上浮 (`staggered fade-up`) 动画。

### 技术实现
*   **路由**: `/blog`
*   **组件库**: 集成 **Element UI** 处理复杂的表单筛选（Cascader, Select, Input）和分页逻辑。
*   **数据模拟**: 包含丰富的中英双语模拟数据，涵盖前端、后端、设计、产品等多个领域。

## � UI-UX-Pro-Max 设计规范

本项目严格遵循 **UI-UX-Pro-Max** 设计标准，这不仅是一套视觉风格，更是对用户体验的极致追求。后续开发**必须**遵守以下原则：

### 1. 视觉美学 (Aesthetics)
*   **玻璃拟态 (Glassmorphism)**: 
    *   **标准**: 使用 `.glass-panel` 类。
    *   **参数**: 背景白/黑半透明，`backdrop-filter: blur(16px)`，带 1px 细微高亮边框和柔和投影。
*   **活力渐变 (Vibrant Gradients)**: 
    *   **文字**: 标题和 LOGO 必须使用 `.gradient-text`，采用蓝紫粉 / 橙黄流体渐变。
    *   **背景**: 避免纯色死板背景，使用动态粒子或柔和光晕。
*   **层级感**: 必须通过阴影 (`box-shadow`) 和 Z轴位移体现元素的空间位置，拒绝扁平化。

### 2. 交互体验 (Interaction)
*   **微交互 (Micro-interactions)**: 
    *   **原则**: 此处无死角。任何可点击元素（按钮、卡片、链接）在 Hover 时必须有视觉反馈（上浮、变色、发光）。
    *   **示例**: 文章卡片 Hover 时，图片轻微放大，卡片整体上浮 5px，阴影加重。
*   **编排动画 (Motion)**:
    *   **原则**: 拒绝生硬的显现。页面元素加载必须带有 `.anim-fade-up` 或类似入场动画。
    *   **节奏**: 列表元素必须使用 `animation-delay` 实现交错显示，形成波浪感。

### 3. 排版与内容
*   **字体**: 首选 `Outfit`, `Inter` 等现代无衬线字体。
*   **间距**: 使用宽松的留白，避免信息过载。

## 🛠 后续开发指南

1.  **新增页面**: 
    *   在 `src/views` 新建 `.vue` 文件。
    *   确保根元素样式包含`position: absolute; width: 100%; height: 100%; overflow-y: auto;`以支持全屏滚动。
    *   内容区域包裹在 `.container` 中以保持对齐。
2.  **样式引用**: 
    *   通用变量（颜色、阴影）在 `src/style/index.css` 的 `:root` 中定义。
    *   **严禁**使用硬编码的颜色值，必须使用 `var(--color-primary)` 等变量。
3.  **组件复用**:
    *   复用 `.glass-panel`, `.gradient-text`, `.anim-fade-up` 等通用类，减少重复 CSS。
