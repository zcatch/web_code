# NEXUS - 数字艺术展示平台

一个专注于技术与艺术融合的创意展示网站，包含丰富的视觉元素和详细的内容描述。

## 🎨 六大内容栏目

### 1. 数字艺术画廊
**视觉元素**:
- 4个 3D 玻璃态几何图形（三角形、立方体、圆柱体、球体）
- 粒子波浪背景
- 技术栈标签

**文字内容**:
- 标题：探索未来设计
- 描述：在这里，代码与艺术交织，创意与技术融合。我们用像素描绘梦想，用算法构建美学，每一个交互都是一次视觉的冒险。
- 技术栈：Vue.js、Three.js、WebGL、GSAP

---

### 2. 视觉实验室
**视觉元素**:
- 4个渐变色卡片网格（粒子系统、流体模拟、光线追踪、形态变换）
- 每个卡片都有独特的渐变动画

**文字内容**:
- 标题：视觉实验室
- 描述：在这个数字实验室中，我们探索视觉的无限可能。从微观的粒子到宏观的宇宙，从静态的几何到动态的流体，每一个实验都是对美学边界的突破。
- 3个实验项目：
  1. **粒子宇宙** - 15,000+ 粒子实时渲染，通过 WebGL Shader 实现的动态波浪效果
  2. **流体诗学** - 基于 Navier-Stokes 方程的流体模拟，GPU 加速计算
  3. **光的旅程** - 实时光线追踪技术，模拟光的传播、反射和折射

---

### 3. 代码诗篇
**视觉元素**:
- 代码编辑器窗口（显示真实的 GLSL Shader 代码）
- 3个哲学卡片

**文字内容**:
- 标题：代码的诗篇
- 描述：代码不仅是逻辑的堆砌，更是思想的表达。在这里，每一行代码都经过精心雕琢，追求简洁、优雅与高效的完美平衡。
- 3个编程哲学：
  1. **简洁至上** - 删除不必要的复杂性，保留核心的本质
  2. **性能为王** - 60 FPS 不是目标，而是底线
  3. **美学驱动** - 技术服务于美学，代码创造艺术

---

### 4. 交互的艺术
**视觉元素**:
- 4个交互卡片，每个都有独特的动画效果：
  - 形态变换的圆形
  - 波浪线条
  - 粒子爆发
  - 渐变流动

**文字内容**:
- 标题：交互的艺术
- 描述：交互不仅是功能的实现，更是情感的传递。每一次点击、每一个悬停、每一帧动画，都在与用户进行一场无声的对话。
- 4个交互理念：
  1. **形态语言** - 几何图形在空间中流转变换，形态的变化传递着状态的转换
  2. **动效节奏** - 缓动函数是精心调校的节奏，每个动画都有其独特的韵律
  3. **反馈机制** - 即时的视觉反馈让用户感知到系统的响应
  4. **色彩情绪** - 色彩不仅是装饰，更是情感的载体

### 5. 技术深度
**视觉元素**:
- 3个 3D 卡片层（带渐变背景）
- WebGL 层：20个浮动粒子动画
- Canvas 层：5条波浪滑动动画
- DOM 层：9格网格脉冲动画
- 左右联动交互

**文字内容**:
- 标题：技术的深度
- 描述：表面的华丽背后，是扎实的技术功底
- 3个技术层详解（每个都有玻璃态卡片）：
  1. **WebGL 渲染层** - 15,000+ 粒子实时渲染，包含统计数据
  2. **Canvas 2D 层** - 2D 特效实现，包含性能指标
  3. **DOM 交互层** - Vue + GSAP 动画引擎，包含组件统计

**交互特效**:
- 悬停左侧卡片，右侧对应卡片高亮
- 悬停右侧卡片，左侧对应卡片放大
- 激活状态显示独特的动画效果
- 玻璃态卡片光扫效果
- 标题：技术的深度
- 描述：表面的华丽背后，是扎实的技术功底。从底层的 WebGL 到上层的 DOM 操作，每一层都经过精心设计和优化。
- 3个技术层详解：
  1. **WebGL 渲染层** - 直接操作 GPU 进行图形渲染，实现 15,000+ 粒子的实时动画
  2. **Canvas 2D 层** - 用于实现代码雨、粒子爆炸等 2D 特效
  3. **DOM 交互层** - 基于 Vue 的响应式系统和 GSAP 的动画引擎
