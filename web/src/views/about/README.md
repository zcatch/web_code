# 关于模块

## 模块说明
个人介绍页面，展示个人技术背景、核心技能和技术沉淀历程。

## 功能特性
- 个人资料卡片
- 核心技术栈展示
- 技术沉淀时间线
- 社交链接
- 响应式设计

## 文件结构
```
about/
├── About.vue          # 关于页面
└── README.md          # 本文档
```

## 页面结构

### 1. 个人资料卡片
**内容:**
- 头像（字母 A）
- 姓名：AIZO | 个人技术空间
- 角色：专注于 Web 技术栈的工程化实践与效率提升
- 个人简介（两段）
- 社交链接（GitHub、Email）

**设计:**
- 玻璃态面板
- 渐变色头像
- 悬停动画效果

### 2. 核心技术栈
**三大类别:**

1. **现代前端**
   - Vue 3 / Composition API
   - TypeScript 高级类型
   - Grid / Flexbox 布局
   - Three.js 交互

2. **后端与存储**
   - Node.js 优化
   - Go 并发模式
   - Redis 分布式锁
   - V8 内存分析

3. **运维与交付**
   - Docker 容器化
   - Nginx 高级配置
   - Shell 自动化备份
   - Linux 系统分析

**设计:**
- 卡片式布局
- 图标 + 标签展示
- 悬停效果
- 渐变色图标背景

### 3. 技术沉淀时间线
**四个阶段:**

1. **当前重点** - 架构设计与性能优化
2. **持续探索** - 容器化与自动化运维
3. **核心里程碑** - TypeScript 与 Vue 3 深度实践
4. **技术起点** - 基础 Web 技术研磨

**设计:**
- 垂直时间线
- 圆点标记（当前项高亮）
- 悬停展开效果
- 渐变色过渡

- 悬停上浮 8px
- 颜色变为主色调
- 光泽扫过效果
- 阴影增强

## 响应式设计

### 断点
- 1024px: 网格布局改为单列
- 768px: 个人资料改为垂直布局
- 576px: 字体和间距缩小

### 移动端优化
- 个人资料居中对齐
- 技能卡片堆叠
- 时间线简化
- 触摸友好的间距

## 样式规范

### 颜色方案
- 主色：粉色渐变
- 辅助色：蓝色、红色、绿色（技能图标）
- 文字：深灰色系
- 背景：浅灰色

### 动画
- 入场动画：fadeUp（0.8s）
- 悬停动画：cubic-bezier(0.23, 1, 0.32, 1)
- 过渡时间：0.3s - 0.5s

### 修改时间线

- 阶段描述内容
- 当前重点标记
