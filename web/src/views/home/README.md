# 首页模块

## 模块说明
网站首页，展示数字艺术与技术融合的创意内容。

## 功能特性
- 7 个全屏展示区域
- WebGL 粒子动画系统
- Three.js 3D 几何图形
- GSAP 滚动动画
- 交互式视觉效果
- 屏幕导航系统

## 文件结构
```
home/
├── Home.vue           # 主页面组件
└── README.md          # 本文档
```

## 七大内容区域

### 1. 代码即艺术
- 标题动画效果
- 技能标签展示
- 粒子波浪背景

### 2. 视觉实验室
- 4个渐变色卡片（粒子系统、流体模拟、光线追踪、形态变换）
- 左右联动交互
- 图标和边框动画

### 3. 代码诗篇
- 代码编辑器窗口
- GLSL Shader 代码展示
- 编程哲学卡片

### 4. 交互艺术
- 4个交互卡片
- 独特的动画效果
- 交互理念展示

### 5. 小程序开发
- 3个手机模型展示
- 旅游平台、社交平台、会员中心
- 完整的小程序界面

### 6. 技术深度
- 3层技术架构（WebGL、Canvas、DOM）
- 左右联动交互
- 渐变卡片和详细说明

### 7. 创作理念
- 3个创作原则
- 旋转几何图形
- 结束语

## 技术实现

### WebGL 粒子系统
- 15,000+ 粒子实时渲染
- 自定义 Vertex/Fragment Shader
- GPU 加速计算

### Three.js 3D 图形
- 4个几何体（三角形、立方体、圆柱体、球体）
- 玻璃态材质
- 旋转和悬浮动画

### GSAP 动画
- ScrollTrigger 滚动触发
- 元素进场动画
- 时间线控制

### 屏幕导航
- 右侧固定导航点
- 悬停显示标题
- 自动跟踪当前屏幕
- 平滑滚动切换

## 性能优化
- 使用 `immediateRender: false` 避免初始隐藏
- 粒子系统使用 BufferGeometry
- 合理使用 requestAnimationFrame
- 避免不必要的重绘

## 响应式设计
- 移动端优化布局
- 触摸事件支持
- 自适应字体大小
- 灵活的网格系统
