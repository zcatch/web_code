# 学习笔记管理方案

## ❓ 问题

如何有效管理学习过程中的技术问答，方便后续复习和回顾？

## ✅ 解答

### 推荐方案：Markdown + Vue 动态展示

针对大量学习内容的管理，采用索引化 + 懒加载的方案。

### 文件结构

```
src/views/learning/data/qa/
├── index.json              # 问答索引（核心）
├── categories/
│   ├── linux/
│   │   ├── qa-001.md
│   │   └── qa-002.md
│   ├── docker/
│   ├── vue/
│   └── ...
```

### 核心特点

**1. 索引化管理**
- 使用 `index.json` 存储所有问答的元数据
- 列表页只加载索引，不加载完整内容
- 支持快速搜索、筛选、排序

**2. 懒加载**
- 详情页才加载完整 Markdown 内容
- 已访问的内容缓存到内存
- 大幅提升性能

**3. 分类体系**
- 一级分类：Linux、Docker、Vue、前端、后端、通用
- 二级分类：每个技术的细分主题
- 标签系统：支持多维度检索

**4. 页面设计**
- 列表页：卡片式展示，显示预览
- 详情页：完整内容渲染
- 支持上一条/下一条快速浏览

### 技术实现

**前端技术栈**
- Vue 3 + Vue Router
- Vite 动态导入
- Markdown 解析（marked + highlight.js）
- 虚拟滚动（可选，数据量特别大时）

**数据流程**
1. 加载 `index.json` 获取问答列表
2. 渲染卡片列表（只显示预览）
3. 点击卡片跳转详情页
4. 动态加载对应的 `.md` 文件
5. 解析并渲染 Markdown

### Markdown 格式规范

```markdown
# 问题标题

## ❓ 问题
具体问题描述

## ✅ 解答
详细解答内容

## 💡 小贴士
相关提示

## 🔗 参考链接
- [文档](url)
```

## 💡 优势

- ✅ Markdown 编辑简单快捷
- ✅ 支持版本控制（Git）
- ✅ 页面自动渲染展示
- ✅ 高性能，支持海量数据
- ✅ 便于导出和分享
- ✅ 支持代码高亮、表格、图片

## 🔗 相关技术

- **Vite**: 现代化构建工具
- **Vue Router**: 路由管理
- **marked**: Markdown 解析
- **highlight.js**: 代码高亮
